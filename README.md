# Austin's Homepage
https://www.austinsehorn.dev

## Stack
- next.js
- typescript
- react-notion-x
- notion API

## Getting Started

Install dependencies:

```bash
npm install
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
